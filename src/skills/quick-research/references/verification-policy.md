# Verification Policy

Verification depth scales with claim weight.

| Claim type | Required support |
|---|---|
| Background context | One source acceptable. |
| Load-bearing factual claim | Two independent sources, **or** one primary source. |
| Numeric value, date, or direct quote | Direct primary source. |
| Predictive / speculative | Cite who is saying it. Do not state as fact. |

## Independence

Two sources count as independent only if they do not derive from the same underlying report. Two outlets quoting the same press release are one source, not two.

## Single-source fallback

If only one source is available within budget for a load-bearing claim, do not silently drop it. Report it with `confidence: low — single source only`.

## Conflicts

Surface, do not smooth.

- Note the disagreement explicitly in the report.
- State which source is closer to primary.
- State your interpretation, marked as inference.

## Out-of-scope evidence

If a load-bearing claim cannot be verified within budget, do not bluff. State the limit and what additional research would be needed.
